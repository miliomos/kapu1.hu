# Kapu1.hu 🌐

A **Kapu1** egy modern, gyors és letisztult startlap a magyar internethez. A klasszikus portálok inspirálták, de a mai kor igényeinek megfelelően Next.js alapon, sötét/világos móddal és reszponzív dizájnnal készült.

## ✨ Funkciók
- **Gyorselérés**: Válogatott linkek a legfontosabb magyar híroldalakhoz, tech és szórakoztató portálokhoz.
- **Téma választó**: Intelligens sötét és világos mód támogatás, amely alapértelmezetten követi a rendszered beállításait.
- **Kereső**: Integrált keresés a weben.
- **Modern Tech Stack**: Next.js 15, React 19 és Tailwind CSS v4.
- **Dinamikus Favicon**: A kód által generált ikon, amely mindig illeszkedik a brandhez.

## 🚀 Telepítés és Futtatás

Ha szeretnéd helyben futtatni a projektet:

1. **Klónozd a tárolót:**
   ```bash
   git clone https://github.com/miliomos/kapu1.hu.git
   cd kapu1.hu
   ```

2. **Telepítsd a függőségeket:**
   ```bash
   npm install
   ```

3. **Indítsd el a fejlesztői szervert:**
   ```bash
   npm run dev
   ```

Nyisd meg a [http://localhost:3000](http://localhost:3000) oldalt a böngésződben.
